JEE Generator

Java Enterprise Entwicklung leicht gemacht! Entwickeln Sie auf Basis einer einfachen Modelldatei eine komplette JEE-Webanwendung. Schauen Sie sich zur Einführung dieses Video an oder lesen Sie in unsere Dokumentation

Die JEE-Generatoren sind in der Lage, eine komplette JEE-Web-Applikation (Java EE 6 oder Java EE 7) aus einem einfachen Modell zu generieren. Der JEE-Generator erzeugt alle Artefakte von der Business Logik angefangen bis zum Presentation Layer samt Deskriptoren. Das Framework wurde mit Xtext realisiert und ist als Eclipse-Plugin verfügbar. Für Java EE 6 und Java EE 7 stehen getrennte Plugins zur Verfügung.

Unterstützte Features

Volle Unterstützung des JEE-Standards

Die JEE-Generatoren erzeugen Code für XHTML, Java Server Faces, EJB und JPA. Eigener Code und Generator-Code kann beliebig gemischt werden.

Lokalisierung und Internationalisierung

Länderspezifische Anpassungen werden auf einfache Art und Weise unterstützt. Die Anwendung benutzt automatisch die im Browser eingestellte Sprache.

Rollenbasierte Sicherheit mit JAAS

Durch rollenbasierte Sicherheit können Benutzer mit unterschiedlichen Rollen unterstützt werden. Jeder sieht in der Oberfläche nur das, was er sehen darf.

Content Assist und Quick Fixes

Der JEE-Generator unterstützt viele Features, um den Umgang zu erleichtern. Er nutzt implizit Features vom Xtext-Framework
